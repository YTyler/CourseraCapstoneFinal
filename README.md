# CourseraCapstoneFinal-Readme.md

#### _Los_Angeles_Neighborhood_Businesses_, _June 2019_

#### By _**Tyler Yates**_

## Description

Sorts LA County neighborhoods (based on zipcode) into 8 groups based on nearby business and locations

## Setup/Installation Requirements

Currently the project can be accessed by downloading or cloning the repository at github.com

The repository can be downloaded/cloned with the link [here](https://github.com/YTyler/CourseraCapstoneFinal)

An in-depth report of the project can be found under "CourseraCapstone.pdf"

"LA_Neighborhoods.PNG" holds a map of all the data points

"LA_Neighborhoods_Clustered.PNG" holds the same data points color-coordinated to their group

"CapstoneNotebook.ipynb" holds a jupyter notebook outlining the python code used to sort the groups
